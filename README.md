### 🎮 Como Usar

<div align="center">

**É super fácil começar!** 🎉

```bash
# 1. Clone o repositório
git clone https://github.com/byamxni/bloomu-dashboard.git

# 2. Entre na pasta
cd bloomu-dashboard

# 3. Abra no navegador (é só um arquivo HTML! ✨)
open index.html
```

**Ou ainda mais fácil:**  
✨ **Acesse diretamente online:** [bloomu-dashboard.vercel.app](https://bloomu-dashboard.vercel.app/)

</div>

### 🎯 Primeiros Passos

1. **✨ Configure seu Perfil**  
   Vá em "Meu Perfil" e personalize suas informações

2. **🎯 Adicione Tecnologias**  
   Marque as tecnologias que você está estudando

3. **📅 Planeje sua Semana**  
   Use o cronograma semanal para organizar seus estudos

4. **🌟 Conecte com GitHub**  
   Adicione seu usuário para ver suas estatísticas
